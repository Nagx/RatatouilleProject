Pour éditer le contenu de l'actualité veuillez suivre les étapes suivantes :
-Aller dans le dossier "contenus" et ouvrir le fichier "actualités.php.
-Editer seulement le texte entre les balises "function titre1() { ?>" et "<?php };".
-Bien suivre les commentaires pour éditer le contenu désiré.
-Sauvegarder les changements.
-Pour modifier les images ouvrez le dossier "images" et remplacez les images désirés :
	actu1.jpg, actu2.jpg, actu3.jpg
-Vos images doivent avoir une dimenssion de 100x100px.
